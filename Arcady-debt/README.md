# Аркадий. Бот-помощник по проверке контрагентов (в процессе)

## Описание функционала
В первоначальном варианте по ИНН оценивает платежеспособность юридического лица: сумму допустимую для авансирования или поставки с отсрочкой на срок до 30 календарных дней (стандартный максимальный срок для поставки под реализацию)

## Состав проекта и функции:
- **Модуль 1** (Private. В процессе). Обработка данных бухгалтерской отчетности. Использует данные бухгалтерской отчетности проекта https://github.com/irlcode/RFSD:
   - Берет данные за последний год отчетности юрлица
   - Очищает данные до необходимых для расчета платежеспособности по моей формуле
   - Расчитывает платежеспособность (сумма на 30 дней, период оборачиваемости кредиторской задолженности в днях)
   - Формирует сводный CSV файл с реестром юрлиц и определенными параметрами
- **Модуль 2**. Определение платежеспособности при отсутствии данных бухгалтерской отчетности. Использует сводный CSV:
   - На данных сводного CSV файла обучает модель определять сумму и срок оборачиваемости кредиторской задоженности
   - Определяет прогнозные значения для тех юрлиц, у кого отсутствует бухгалтерская отчетность
   - Очищает и формирует итоговый CSV файл с данными ИНН, и расчетных показателей
- **Модуль 3**. Бот "Аркадий":
   - Получает на вход ИНН
   - Отдает значения предельной дебиторской задолженности на срок до 30 календарных дней
   - Определяет, можно ли авансировать или давать отсрочку до 30 календарных дней, отдает рекомендацию
 
## В перспективе:

- Оцифровка моей методики оценки рисков сотрудничества **SCPV**
- Формирование сводного отчета с рекомендациями о сотрудничестве
